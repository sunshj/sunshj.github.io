Linux服务器安装Typecho

一、环境安装

1. 一键安装宝塔面板

中间需要进行 “确认”，输入y 回车即可

2. 安装完成面板信息

3. 登录宝塔面板

打开浏览器进入面板，会提示安装 LNMP,将PHP改为7.1版本，选择一键安装

二、创建网站

添加网站，并同时新建数据库。记录下数据库的用户名 和密码。

三、安装 Typecho

进入 Typecho 官网，下载安装包。建议下载正式版。

打开网站管理页面，进入网站目录，把该目录中的现有文件全部删除。然后把 Typecho 文件压缩包上传到网站目录中并解压，解压后出现 build 文件夹。

把 build 文件夹中的文件全部复制到网站目录文件夹中，然后删除 build 文件夹 和 Typecho 压缩包。

访问http://ip或域名/install.php

按提示完成 Typecho 配置
