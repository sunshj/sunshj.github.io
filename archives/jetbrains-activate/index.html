以下操作以Intellij IDEA 为例


一、首先进入IDEA，先选择试用30天

二、安装插件

1.打开file ——> settings


2.Plugins——> Install Plugin from Disk


3.选择文件夹中的BetterIntelliJ.zip, 点击OK



4.重启IDEA

！！！此步最重要！！！

5.查看是否安装成功

Help ——> Edit Custom VM Options

显示此行即安装成功


三、激活


1.选择 Help ——> Register


2.添加激活码

复制 激活补丁key.txt 的内容，选择添加激活码 (Activation code) ，输入激活码点击激活 (Activate)

四、注意事项

激活完成后，不要删除、移动文件夹，否则激活会失效！
此激活方法同样适用于JetBrains家族其他产品，如PyCharm
